# Large-Scale CVRPTW Optimization for Annual Fleet Cost Benchmarking

This repository presents an optimization project focused on solving a **large-scale Capacitated Vehicle Routing Problem with Time Windows (CVRPTW)** for fleet planning and annual transportation cost benchmarking.

The project is designed to improve delivery efficiency across a large store network by building practical routing and clustering workflows that support better truck route design, lower mileage, and more structured regional planning. The repository includes optimization notebooks as well as a competition report documenting the overall solution approach. 

## Project Overview

Large-scale fleet planning requires balancing multiple real-world constraints at the same time, such as:

- vehicle capacity limits
- store delivery schedules
- fixed service windows
- distribution center coordination
- route feasibility over a broad geographic region

This repository explores a scalable approach to routing optimization by combining:

- **CVRPTW-based route planning**
- **store clustering for regional structuring**
- **distance-matrix-driven optimization**
- **visual analysis through interactive mapping**

The goal is to benchmark and improve annual fleet cost and routing efficiency for a multi-location delivery network.
